# Куда подключать провода

Перед подключением выключить питание.

## LCD 16×2

Подключается к разъёму `I2C` на плате Arduino:

```text
GND → GND
VCC → 5V
SDA → SDA
SCL → SCL
```

## ИК-приёмник

```text
VCC → 5V
GND → GND
OUT → D8
```

## Сервопривод TowerPro MG996R

```text
Красный провод (+) → +5,86 В после предохранителя F1 3 А
Коричневый или чёрный провод (−) → минус первого XL4015E1
Жёлтый или оранжевый сигнальный провод → D9 Arduino
```

Конденсаторы `2400 мкФ` и `100 нФ` подключены параллельно между плюсом и
минусом питания сервопривода после предохранителя.

## Весовой модуль HX711

```text
VCC → 5V
GND → GND
DT или DOUT → D3
SCK или CLK → D2
```

## Тензодатчик

Подключается к модулю HX711:

```text
E+ → E+
E− → E−
A+ → A+
A− → A−
```

## Мотор N20 и MOSFET-модуль

Зелёный двухконтактный разъём MOSFET-модуля:

```text
PWM → D11
GND → GND Arduino
```

Синий трёхконтактный разъём MOSFET-модуля:

```text
Клемма «− источника» → минус третьего XL4015E1
Клемма «+ источника / + мотора» → скрутка двух красных проводов:
  плюс третьего XL4015E1 + плюс мотора N20
Клемма «− мотора» → минус мотора N20
```

Третий XL4015E1 настроен на `12 В`.

## RGB-светодиод

Каждый цвет подключается через отдельный резистор `220 Ом`:

```text
B / синий → резистор 220 Ом → D5
G / зелёный → резистор 220 Ом → D6
R / красный → резистор 220 Ом → D7
общий вывод RGB → GND
```

## Краткая таблица

| Провод | Контакт Arduino |
|---|---|
| `HX711 SCK/CLK` | `D2` |
| `HX711 DT/DOUT` | `D3` |
| `RGB B через 220 Ом` | `D5` |
| `RGB G через 220 Ом` | `D6` |
| `RGB R через 220 Ом` | `D7` |
| `OUT ИК-приёмника` | `D8` |
| `сигнал сервопривода` | `D9` |
| `PWM мотора` | `D11` |
| `LCD SDA` | `SDA` |
| `LCD SCL` | `SCL` |
