# Инструкция по пульту дозатора

## Главное меню

После включения дозатор показывает меню выбора экрана:

```text
Select screen
> Weight input
```

| Кнопка | Действие |
|---|---|
| `PREV` | Выбрать предыдущий экран |
| `NEXT` | Выбрать следующий экран |
| `PLAY/PAUSE` | Открыть выбранный экран |

Экраны идут по кругу:

```text
Weight input -> Angle setup -> Calibration -> Work -> Weight input
```

## Экран ввода веса

| Кнопка | Действие |
|---|---|
| `0-9` | Ввести цифру в выбранную позицию |
| `PREV` | Сдвинуть курсор влево |
| `NEXT` | Сдвинуть курсор вправо |
| `EQ` | Обнулить все цифры |
| `PLAY/PAUSE` | Сохранить ненулевой вес и открыть меню |

## Экран настройки заслонки

| Кнопка | Действие |
|---|---|
| `VOL-` | Уменьшить текущее значение на 1 |
| `VOL+` | Увеличить текущее значение на 1 |
| `PREV` | Перейти к предыдущей настройке |
| `NEXT` | Перейти к следующей настройке |
| `PLAY/PAUSE` | Сохранить настройки, закрыть заслонку и открыть меню |

Этапы настройки:

```text
OPEN ANGLE <-> CLOSED ANGLE <-> SLOW ANGLE <-> STOP WEIGHT
```

`STOP WEIGHT` — остаток веса в граммах, при котором основная подача
закрывает заслонку и переходит к точной досыпке.

## Экран калибровки весов

| Кнопка | Действие |
|---|---|
| `EQ` | Обнулить пустые весы |
| `PLAY/PAUSE` | Открыть меню |

Ход калибровки:

```text
Press EQ
to zero
```

После обнуления:

```text
Put 1kg
wait...
```

Когда груз 1 кг зафиксирован, коэффициент сохраняется в EEPROM:

```text
OK saved
PLAY to menu
```

## Рабочий экран

| Кнопка | Действие |
|---|---|
| `CH` | Обнулить весы и запустить дозирование |
| `EQ` | Только обнулить весы |
| `PLAY/PAUSE` | Остановить дозирование и открыть меню |
